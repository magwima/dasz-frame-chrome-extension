# dasz-frame-chrome-extension
panography photo editor
**Installation steps:**

Download and unzip the file to a permanent folder (e.g. ~/facet-extension — don't delete it after)
Open Chrome → go to chrome://extensions
Toggle on Developer mode (top-right)
Click "Load unpacked"
Select the unzipped facet-extension folder
The FACET icon appears in your Chrome toolbar — click it to open
