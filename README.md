# DASZFRAME — AI Fragment Portrait Studio v1.1
### Chrome Extension — Full Tab Studio

DASZFRAME — Transform any portrait into AI-directed fragmented artwork.

---

## Install

1. Unzip this folder to a permanent location (e.g. ~/facet-extension)
2. Go to chrome://extensions
3. Enable Developer mode (top right)
4. Click "Load unpacked" → select the facet-extension folder
5. Click the DASZFRAME toolbar icon → opens as a full browser tab

---

## View Modes (always in the header)

| Mode | Description |
|------|-------------|
| **Split** | Default — left controls, canvas center, right layers/props |
| **Canvas** | Full canvas focus, panels hidden (use toolbar toggles ◧◨ to bring back) |
| **Panels** | All three sections spread wide — ideal on large monitors |

## Panel Controls

- Click **←/→** on any panel header to collapse/expand it
- Collapsed panels show a vertical label — click it to re-expand
- Toolbar buttons **◧ ◨** toggle left/right panels independently

## Canvas Sizes

Switch between SM / MD / LG / WIDE in the Controls panel. Fragments rescale automatically.

## Features

- 6 layout modes · 6 styles · AI art direction · Full manual editing
- Per-fragment style override, flip, opacity, rotation, scale
- Layer visibility, z-order, duplicate, delete
- Layout variations (6 previews, click to apply)
- Export PNG/JPG at Standard, HD (2×), 4K (4×)
